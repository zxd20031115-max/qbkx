# 情報科学 I 学习工作台

这是一个用于复习「情報科学 I」和「基本情報技術者」基础内容的静态网页。

## 内容

- 课程地图：第 1 回到第 8 回讲义内容
- 后续预备：网络、数据库、信息安全、管理基础
- 章节图解：集合、逻辑回路、流程图、CPU、接口、CMOS 等
- 中日概念词典
- 进制、补数、逻辑、缓存、通信、稼働率计算工具
- 讲义原题库：課題、練習問題、復習問題
- IPA 官方题库入口

## GitHub Pages 发布方法

1. 在 GitHub 新建仓库，例如 `info-science-site`
2. 上传本文件夹中的所有文件
3. 打开仓库 `Settings` → `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，Folder 选择 `/root`
6. 保存后等待 GitHub Pages 生成网址

生成的网址通常类似：

```text
https://你的用户名.github.io/info-science-site/
```

## 文件

- `index.html`：网页结构
- `styles.css`：样式
- `app.js`：内容数据和交互逻辑
- `.nojekyll`：让 GitHub Pages 直接发布静态文件
